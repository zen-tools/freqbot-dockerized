get_lang

показывает язык, который бот использует в текущей конференции. Изменить язык можно командой .set_lang

lang_list

показывает список доступных языков. Синтаксис: .lang_list

set_lang

выбор языка бота в текущей конференции. синтаксис: .set_lang foo, например: .set_lang ua. Список доступных языков можно посмотреть командой .lang_list
